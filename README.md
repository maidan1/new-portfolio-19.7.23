## איך להריץ את הפרוייקט

אם אתה מעוניין להריץ ולבדוק את הפרוייקטים בעצמך, כדאי לעקוב אחרי ההוראות הבאות:

1. הורד את הקובץ ZIP או המאגר מ- GitHub.

3. עכשיו תוכל לראות ולעיין בפרוייקטים שלי ובדפי הנחיתה שלהם.
